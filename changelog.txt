# Changelog

All notable changes to ILP - Internal Linking Pro are documented in this file.
The project follows Semantic Versioning.

## [2.1.0] - 2026-06-06

### Added

- Responsive administration layouts for laptop, tablet, and mobile screens.
- Modern icons and visual states for the administration tabs and controls.
- Cluster URLs below cluster titles on the dashboard, opening in a new tab.
- `Accueil` as the fallback title for unnamed pages in the content list, most-linked list, and dashboard cluster list.

### Changed

- Redesigned dashboard statistics, score, analyzed-content, most-linked, suggestions, and cluster sections.
- Modernized the cluster table, conflict overview, summary cards, and insights.
- Redesigned the graph node details panel.
- Harmonized borders, shadows, spacing, buttons, filters, and cards across administration tabs.
- Improved Gutenberg suggestion ranking and refreshed candidates from current published content.
- Preserved the existing desktop layout above 1440 pixels while improving smaller viewports.
- Reduced administration asset loading outside the graph view.

### Security

- Added direct-access protection to the link-suggestions administration view.
- Extended uninstall cleanup for dynamic Gutenberg suggestion transients.

## [2.0.2] - 2026-05-31

### Changed

- Maintenance release preceding the 2.1 responsive interface update.
