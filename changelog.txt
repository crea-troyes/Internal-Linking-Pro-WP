# Changelog

All notable changes to ILP - Internal Linking Pro are documented in this file.
The project follows Semantic Versioning.

## [2.5.8] - 2026-06-01

### Added

- GitHub and WordPress.org publication files.
- GPL-2.0-or-later license metadata.
- Third-party notice for the bundled `vis-network` library.
- PHPCS configuration for WordPress Coding Standards.

### Changed

- Loads the vendored `vis-network` library only on the graph tab.

### Security

- Added the missing direct-access guard to the link-suggestions administration view.
- Extended uninstall cleanup to remove dynamic Gutenberg suggestion transients.

## [2.5.7] - 2026-06-01

### Changed

- Gutenberg suggestions now prioritize direct and close title matches before context.

## [2.5.6] - 2026-06-01

### Changed

- Refreshed Gutenberg candidates from the current published-content catalogue.
- Preserved cached graph metrics while filtering obsolete catalogue entries.

## [2.5.5] - 2026-06-01

### Changed

- Reduced contextual influence in Gutenberg suggestions.
- Prioritized the exact selected expression and direct title relevance.

## [2.5.0] - 2026-06-01

### Added

- Gutenberg internal-link suggestion panel.
- SEO cannibalization conflict analysis.
- French and Spanish translations.
